# Typing-Speed-Tester
A program to test out typing speed in a type of a game.
You can use this program to type out paragraphs from movies, books, songs etc. Which creates a fun and interactive experience for the user.
FEATURES INCLUDE-
Login Page.
Option to select category.
Option to choose a random category.
Option to choose a random paragraph from a random category.
Progress bar
WPM
Accuracy
Time taken to type the specific quote.
Personal Best
